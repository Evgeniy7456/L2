Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Вывод: [77 78 79]

a - массив из 5 элементов
b - слайс из элементов массива a с 1 индекса по 3 включительно

```
