Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Вывод:
2
1

В функции test сразу указано название возвращаемой переменной и переменной x сразу присваивается значение 0.
Ниже оно изменяется на 1. Перед завершением работы функции, т.е. перед возвращением результата выполняется
анонимная функция, которая увеличивает значение x на 1. В итоге результат равен 2.

В функции anotherTest не указано название возвращаемой переменной. Поэтому создается временная переменная.
Далее создается переменная x и ей присваивается значение 1. При выполнении строки return x временной
переменной присваивается значение x. Перед завершение работы функции выполняется анонимная функция, которая
увеличивает значение x на 1, но это никак не влияет на временную переменную значение которой будет возвращено.

```
